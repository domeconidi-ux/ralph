## Codebase Patterns
- `Amount` component in `components/ui/Typography.tsx` handles it-IT formatting automatically (pass `value` prop)
- `shadows` is exported from `constants/tokens.ts` alongside `colors`
- Dashboard (`app/(protected)/(tabs)/index.tsx`) uses `useExpenses()` which returns `expenses: ExpenseRow[]` with `amount: number` and `date: string` (YYYY-MM-DD format)
- `ExpenseRow.category` is a plain `string` (not nullable) — safe to use as Record key directly
- `getCategoryById(id)` from `@/constants/categories` returns `Category | undefined`; fallback: emoji `'📦'`, nome = catId
