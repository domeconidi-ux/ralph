## Codebase Patterns
- Font packages from @expo-google-fonts register with underscore names (e.g., `Nunito_400Regular`), but tokens.ts uses hyphen names (e.g., `'Nunito-Regular'`). When using `fontFamily` in styles, use the keys from useFonts() call, not tokens.ts font values directly.
- Bash tool does NOT return output directly in MINGW64 session â€” use `run_in_background: true` + redirect to file + Read the file to see output (e.g., `cmd > output.txt 2>&1`).
- Expo Router typed routes: use `"/"` (not `"/(protected)/(tabs)/"`) to navigate to the tabs index. Valid paths have no trailing slash and match exactly what's in `.expo/types/router.d.ts`.
- lucide-react-native requires `--legacy-peer-deps` when React 19 is used (peer dep declares ^16/17/18 only). react-native-svg is a required peer dep too.
- Custom brand colors (coral, teal, navy, offWhite, etc.) must be added to tailwind.config.js `theme.extend.colors` before they can be used as NativeWind classes (e.g., `bg-coral`, `text-coral`, `border-coral`, `bg-offWhite`).
- When removing indigo from public pages, also check `app/(public)/join/[code].tsx` â€” it is part of the public layout tree and must also be reskinned.
